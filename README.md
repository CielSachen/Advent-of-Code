# Advent of Code

My solutions for [Advent of Code](https://adventofcode.com/) problems throughout the years. I only
solve problems in Python, but may consider other languages in the future. The solutions are not
guaranteed to be efficient, because I’m an amateur in programming.

Only solutions for days solved **during** the event will be added. I don't solve the remaining days
outside of the event.
